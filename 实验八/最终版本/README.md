# SRM + STDP 模式识别实验

## 项目概述

本项目实现了一个基于 **SRM（Spike Response Model）** 单神经元输出与 **STDP（Spike-Timing-Dependent Plasticity）** 学习的模式识别仿真系统。

输入层由 2000 个泊松神经元组成，其中 50% 的神经元参与一个 50 ms 的重复模式。输出神经元通过 STDP 学习增强对模式脉冲的响应，最终能够在模式出现时快速、可靠地发放。

## 模块说明

### 1. main.py（主程序入口）
- **阶段1**：运行 450 s 主仿真，生成输入脉冲、运行 SRM+STDP、评估性能、绘制图1–3。
- **阶段2**：对两个关键参数（模式频率、初始权重）进行敏感性扫描，每个点重复 10 次试验，绘制图4。
- 直接运行：`python main.py`

### 2. config.py（全局参数配置）
集中存放所有实验参数，包括：
- 网络规模与时间参数（N=2000, T_sim=450s）
- 输入脉冲参数（背景率 52 Hz，自发率 10 Hz，模式频率 0.25）
- SRM 神经元参数（τ_m=10 ms, τ_s=2.5 ms, 阈值 500）
- STDP 参数（a₊=0.03125, τ₊=16.8 ms, a₋=0.85a₊, τ₋=33.7 ms）
- EPSP 核归一化预计算常数

### 3. input_generator.py（输入脉冲生成模块）
核心函数 `generate_input_spikes(...)` 完成以下步骤：
1. 为每个输入神经元生成固定背景率（52 Hz）的泊松脉冲。
2. 随机选取 50% 神经元作为模式神经元群。
3. 从背景脉冲中截取 50 ms 片段作为模式模板。
4. 按指定频率将带抖动（σ=1 ms）的模式模板插入时间轴。
5. 添加自发活动噪声（10 Hz）。
6. 构建全局时间索引，支持仿真时的快速向量化访问。

### 4. neuron_stdp.py（输出神经元与 STDP 学习模块）
包含两个核心函数：

- **`simulate(...)`**：SRM 主循环
  - 逐 ms 更新 EPSP 双指数状态（u, v）。
  - 计算膜电位 p = η(t) + EPSP(t)。
  - 发放判定：p ≥ 阈值 且 不在不应期内。
  - 发放后触发 STDP 权重更新：
    - **LTP**：输入脉冲在输出脉冲之前（pre → post），权重增加。
    - **LTD**：输入脉冲在输出脉冲之后（post → pre），权重减少。
  - 权重裁剪到 [0, 1]。

- **`evaluate_trial(...)`**：性能评估
  - 取最后 150 s 评估窗口。
  - 统计模式命中率、平均潜伏期、误报次数。
  - 成功标准：潜伏期 < 10 ms、命中率 ≥ 98%、无误报。

### 5. plotting.py（绘图与结果分析模块）
提供 4 个绘图函数，对应实验要求的 4 张图：

| 函数 | 输出文件 | 内容说明 |
|------|---------|---------|
| `plot_fig1` | `fig1_input_pattern.png` | 输入脉冲栅格图 + 群体/单神经元发放率 |
| `plot_fig2` | `fig2_membrane_potential.png` | 膜电位轨迹与阈值（初期/中期/后期） |
| `plot_fig3` | `fig3_latency.png` | 潜伏期随发放次数下降曲线 |
| `plot_fig4_two_params` | `fig4_sensitivity.png` | 参数敏感性扫描（成功率 vs 参数值） |

## 关键设计说明

1. **全局索引加速**：`input_generator.py` 中构建的 `time_start` 数组将脉冲数据展平为连续存储，仿真循环中通过切片 `all_neurons[idx0:idx1]` 实现 O(1) 访问，避免逐神经元列表查找。

2. **STDP 窗口截断**：LTP 和 LTD 分别限制在 `7τ₊` 和 `7τ₋` 内，避免远距离脉冲产生无意义更新。

3. **参数扫描策略**：扫描时使用 200 s 缩短仿真，每个参数值重复 10 次取平均，平衡统计可靠性与计算成本。

4. **中文支持**：`config.py` 中配置了多平台中文字体回退列表，确保图表标题在不同系统下正常显示。
